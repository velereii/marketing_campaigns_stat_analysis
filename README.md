# Анализ выборочных совокупностей в R

## Описание проекта
Статистический анализ продаж в розничной сети с оценкой эффективности трёх промо-кампаний, проверкой гипотез, построением регрессионных моделей и формированием управленческих рекомендаций. Этот проект выполнен в рамках курса «Математическая статистика» и направлен на практическое применение статистических методов анализа данных.  

## Используемые методы
• Выявление выбросов (методы Тьюки и Z-оценок)  
• Проверка нормальности распределения (Шапиро–Уилк, Крамер–фон Мизес)  
• Проверка статистических гипотез (t-тест, критерии Краскела–Уоллиса и Вилкоксона)  
• Построение парной и множественной линейной регрессии  
• Интерпретация результатов и выработка управленческих решений
